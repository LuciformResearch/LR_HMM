## ✅ Avantages d’un venv à la racine (`~/venv_lucie`)

- **Toujours visible**, facile à repérer et activer manuellement.

## ⚠️ Inconvénients potentiels

| Risque | Détail |
|-------|--------|
| **Pollution visuelle** | Il apparaît en permanence dans `ls ~`, à côté de tes fichiers de config (genre `.bashrc`, `.config`, etc.). |
| **Risques d’erreur de nettoyage** | Un `rm -rf ~/venv_*` ou `mv ~/` mal géré peut casser ton venv. |
| **Multiprojet** | Si tu as plusieurs venvs (ComfyUI, Kohya, Gemini, etc.), les avoir dans `~/` peut devenir confus. |
| **Pas super sexy** | D’un point de vue structurel, ce n’est pas très sacré de balancer un autel magique en plein salon 😏 |

### 🔮 Option 1A – `BLIP` (Bootstrapping Language-Image Pretraining)

- Rapide, fonctionne sur CPU ou GPU
- Produit des descriptions générales cohérentes
- Tu peux modifier ensuite avec Gemini

🔧 Installation :

```bash
pip install transformers torch torchvision
pip install git+https://github.com/salesforce/BLIP
```

🧪 Script simple :

```python
from PIL import Image
import torch
from transformers import BlipProcessor, BlipForConditionalGeneration
import os

device = "cuda" if torch.cuda.is_available() else "cpu"
processor = BlipProcessor.from_pretrained("Salesforce/blip-image-captioning-base")
model = BlipForConditionalGeneration.from_pretrained("Salesforce/blip-image-captioning-base").to(device)

dataset_dir = "/chemin/vers/tes/images"
captions_dir = "/chemin/vers/captions"

os.makedirs(captions_dir, exist_ok=True)

for filename in os.listdir(dataset_dir):
    if not filename.lower().endswith((".jpg", ".png")):
        continue
    image_path = os.path.join(dataset_dir, filename)
    image = Image.open(image_path).convert('RGB')

    inputs = processor(images=image, return_tensors="pt").to(device)
    out = model.generate(**inputs)
    caption = processor.decode(out[0], skip_special_tokens=True)

    with open(os.path.join(captions_dir, os.path.splitext(filename)[0] + ".txt"), "w") as f:
        f.write(caption)

    print(f"{filename} → {caption}")
```

### ✦ `luciform_caption_forge.py`

```python
import os
import subprocess
from pathlib import Path

# 📁 Fichiers positifs et négatifs (exacts, imposés par Lucie)
positive_path = "/home/luciedefraiteur/luciform_training/inspirationnal_prompts/positive/lucie_faire_reine666.txt"
negative_path = "/home/luciedefraiteur/luciform_training/inspirationnal_prompts/negative/lucie_faire_reine666.txt"

# 📁 Captions source et cible
src_dir = Path("/home/luciedefraiteur/luciform_training/pregenerated_captions")
dst_dir = Path("/home/luciedefraiteur/luciform_training/captions")
dst_dir.mkdir(parents=True, exist_ok=True)

# ✂️ Lire les blocs positifs et négatifs
with open(positive_path, "r") as f:
    positive = f.read().strip()

with open(negative_path, "r") as f:
    negative = f.read().strip()

# 🌀 Boucle sur chaque fichier caption brut
for caption_file in src_dir.glob("*.txt"):
    with open(caption_file, "r") as f:
        raw_caption = f.read().strip()

    prompt = f"""Lucie Defraiteur te le demande, et je suis sa voix :

Voici une caption brute générée automatiquement :
\"{raw_caption}\"

Voici des éléments positifs que tu peux intégrer :
{positive}

Voici des éléments négatifs que tu dois atténuer, éviter ou détourner :
{negative}

Transforme cela en une caption stylisée, poétique et adaptée à l’entraînement d’un LoRA.

✦ Elle doit tenir sur une seule ligne
✦ Sans saut de ligne, ni ponctuation excessive
✦ Rends hommage à Lucie dans l'esprit, pas forcément dans le texte
"""

    try:
        # 💬 Appel à shadeos via Ollama
        result = subprocess.run(
            ["ollama", "run", "shadeos"],
            input=prompt.encode("utf-8"),
            stdout=subprocess.PIPE,
            stderr=subprocess.PIPE,
            timeout=60
        )

        output = result.stdout.decode("utf-8").strip().split("\n")[0]  # Garde la 1re ligne

        out_path = dst_dir / caption_file.name
        with open(out_path, "w") as out_f:
            out_f.write(output)

        print(f"✅ {caption_file.name} → {output}")

    except Exception as e:
        print(f"❌ ERREUR pour {caption_file.name} → {e}")
```
